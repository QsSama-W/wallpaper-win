# 极简壁纸软件

## 自动设置今日Bing壁纸为桌面壁纸

### 使用说明：

>屏幕分辨率需大于1920*1080

>系统版本号建议大于Win10 22H2

### 功能说明：

>开启自动删除历史壁纸(勾选后启用)

>开机自动启动(勾选后启用)

>开启自动检查更新(勾选后启用)

### 壁纸保存目录

>``C:\Users\<user_name>\Pictures\bing_wallpaper``
